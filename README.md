🚀 **Projects:**

- **[DevScope](https://github.com/bereketlemma/DevScope)** : `Python` `GCP` `BigQuery` `Pub/Sub` `Vertex AI` `Cloud Run` `React` `TypeScript`
  - Built a distributed engineering analytics platform that mines GitHub repository data via the GitHub API to surface PR velocity, review latency, and code churn for engineering teams.
  - Streams events through Cloud Pub/Sub into BigQuery for sub-second querying. Vertex AI time series anomaly detection on Cloud Run flags productivity regressions with 95% precision.

- **[Network Intrusion Detection System](https://github.com/bereketlemma/network-intrusion-detection-system)** : `Python` `Scikit-learn` `Random Forest` `Streamlit`
  - Built a real-time network intrusion detection system in Python that classifies live traffic across 7 attack categories including DoS, DDoS, port scanning, and brute-force.
  - Trained a Random Forest classifier on 2.5M+ CICIDS2017 network flow records. Applied undersampling to handle severe class imbalance, achieving 97.47% accuracy and 98.21% F1-score on unseen test data. Results visualized in a live Streamlit dashboard.

- **[CPU Scheduler Simulator](https://github.com/bereketlemma/CPU-Scheduler-Simulator)** : `C` `PlanetLab Traces`
    - Implemented FCFS, Round Robin, and MLFQ scheduling from scratch in C and benchmarked them against 1,000+ real PlanetLab distributed VM workload traces across multiprocessor configurations.
    - Measured throughput, CPU utilization, turnaround time, and response time across all three algorithms to identify the optimal strategy per workload type.
